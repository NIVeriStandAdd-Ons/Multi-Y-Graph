## IP Name ##

**Multi-Y Graph** is a workspace object which enables the user to plot multiple Y channels verses a single X channel.

### LabVIEW Version ###

LabVIEW 2013.

### Built Availability ###

No builds are available at this time.

### Quality, Limitations ###

This IP has been in use since 2013.

### Dependencies ###

None.
